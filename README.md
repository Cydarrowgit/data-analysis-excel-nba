# Visualisation de données NBA avec Excel
Analyse et visualisation de données NBA avec Excel, réalisée dans le cadre d’un projet OpenClassrooms.

## Contexte
Projet réalisé dans le cadre du bootcamp Analyse de données d’OpenClassrooms.  
Le projet consistait à analyser une base de données NBA afin de produire des analyses et visualisations compréhensibles pour une équipe de club.

## Objectif
Comprendre un jeu de données inconnu, analyser des statistiques NBA et présenter les principaux enseignements à l’aide d’un support clair et visuel.

## Données
- Base de données NBA fournie (statistiques joueurs et équipes)
- Dictionnaire des données intégré au fichier Excel

## Outils
- Microsoft Excel  
- Tableaux  
- Tableaux croisés dynamiques  
- Graphiques  
- Formules Excel  
- Mise en forme conditionnelle

## Ce que j’ai fait
- Analyse et compréhension de la structure de la base de données  
- Identification et interprétation des différentes variables à l’aide du dictionnaire des données  
- Reproduction et adaptation d’analyses existantes à partir d’un exemple fourni  
- Création de tableaux, tableaux croisés dynamiques et graphiques  
- Prise en compte des critères d’accessibilité dans les visualisations  
- Préparation d’un support de présentation synthétisant les analyses réalisées

## Résultat
Un support de présentation clair et structuré permettant de comprendre rapidement les principales statistiques et tendances issues des données NBA, destiné à un public non technique.
